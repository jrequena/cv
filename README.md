### Sistema de Gestión de Reservas de Hotel

Plataforma web para la gestión de reservas hoteleras, permitiendo controlar la disponibilidad de habitaciones, evitar conflictos de reservas y optimizar la ocupación.

**Características principales:**
- Validación de disponibilidad por rango de fechas
- Prevención de overbooking mediante control de concurrencia
- Gestión de habitaciones, clientes y reservas
- Modificación y cancelación de reservas
- Control de estados (pendiente, confirmada, cancelada)

**Tecnologías:**
- PHP
- Angular
- MySQL
- APIs REST
